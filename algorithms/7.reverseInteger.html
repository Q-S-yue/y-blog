# 整数反转-leecode第五题

# 题目说明

给你一个 32 位的有符号整数 x ，返回将 x 中的数字部分反转后的结果。

如果反转后整数超过 32 位的有符号整数的范围 [−231, 231 − 1] ，就返回 0。

假设环境不允许存储 64 位整数（有符号或无符号）。

# 前置知识

js中数据存储形式，与32和64整数有什么关系，有符号和无符号有什么关系，溢出处理

位运算、等等

js中数据存储形式

# 题解

第一种，就是强制反转

以及，判断负数和边界情况

第二种：位运算 通过取余得到的值作为 res 的第一位 依次类推 每一次都要给 res 乘 10 移位 最后再让 x 除以 10，利用 ~~ 来取整

var reverse = function (x) { let res = 0; while (x) { res = res * 10 + x % 10 // 判断越界 if (res > Math.pow(2, 31) - 1 || res < Math.pow(-2, 31)) return 0; // 对 x 进行取整，只取整数部分 x = ~~(x / 10); } return res };
