# 字符串转化整数(atoi)

leecode第八题

首先分析题意，

myAtoi需要满足要求：

丢弃空格

读取符号

读到非数字字符或结尾

数字转为整数

在32位有符号整数范围[−2^31, 2^31 − 1]中，否则固定为边界值

（其实一个算法解法，最好的就是能归纳总结为一个思想）

解法一：笨方法

处理多种情况，比较臃肿

这种方法，根据不同的颗粒度，有多种方法，

暂时归纳这几种

解法二：parseInt

参考地址：https://leetcode-cn.com/problems/string-to-integer-atoi/solution/javascripttou-ji-qu-qiao-wu-xu-si-kao-yi-kan-jiu-h/

解法三：正则

总结理解题目要求是为了匹配带+或-，后面跟若干位数字的字符串，正则可以满足需求

var myAtoi = function(s) {
    const re = new RegExp(/^(-|\+)?\d+/);
    let str = s.trim().match(re);
    let res = str ? Number(str[0]) : 0;
    return res >= 0 ? Math.min(res, 2**31 - 1) : Math.max(res, -(2**31))
};

dfa：DFA，全称 Deterministic Finite Automaton 即确定有穷自动机：从一个状态通过一系列的事件转换到另一个状态，即 state -> event -> state。计算机操作系统中的进程状态与切换可以作为 DFA 算法的一种近似理解
